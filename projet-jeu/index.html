LE DONJON DU FROID
Bienvenue dans le terrible dongeon aux terribles enigmes... Il fait froid, car le dongeon n'est pas chauffe. Sortez le plus vite possible, en repondant aux questions qui vous seront posees.
Cliquez sur Start pour lancer le jeu. Deplacez vous avez les fleches, et utilisez le clavier. Bonne chance...
